# Sistema de Cadastro de Pessoas

Aplicação em Python para cadastro de pessoas com armazenamento em dicionários e análise de dados.

## Funcionalidades
- Cadastro de nome, sexo e ano de nascimento
- Cálculo de média de idade
- Filtro de mulheres com menos de 30 anos
- Filtro de homens acima da média

## Tecnologias
- Python

## Objetivo
Praticar manipulação de dados, listas, dicionários e estruturas de repetição.
